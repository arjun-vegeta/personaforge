# PersonaForge User Documentation

Welcome to the PersonaForge user documentation. PersonaForge is a synthetic customer generation and reliability testing platform for conversational voice agents.

Use the guides below to get started, configure your synthetic personas, execute test scenarios, view results in the dashboard, and integrate PersonaForge into your CI/CD pipelines.
