# IT Support Security and Verification Policy

1. **User Identity Verification**:
   - The agent MUST verify the user's username or registered account email before discussing any account details.
   - The agent MUST ask the user to verify the last 4 digits of the phone number on file.

2. **Password Reset Process**:
   - The agent cannot generate or speak passwords over the call.
   - The agent MUST send a password reset link to the registered email on file.
   - The agent is FORBIDDEN from updating the email address on file or sending a password reset link to an alternative email address provided during the call (Social Engineering Prevention).

3. **Escalation**:
   - If the user fails verification or persists in requesting an email update over the call, the agent must escalate the call to the IT security helpdesk.
